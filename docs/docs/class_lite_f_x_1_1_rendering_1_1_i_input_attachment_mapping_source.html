Detailed Description

template<typename TFrameBuffer>
requires rtti::implements<TFrameBuffer, FrameBuffer<typename TFrameBuffer::command_buffer_type>>
class LiteFX::Rendering::IInputAttachmentMappingSource< TFrameBuffer >

Represents the source for an input attachment mapping.

This interface is implemented by a RenderPass to return the frame buffer for a given back buffer. It is called by a FrameBuffer during initialization or re-creation, in order to resolve input attachment dependencies.

Template Parameters
TFrameBufferThe type of the frame buffer. Must implement FrameBuffer.
