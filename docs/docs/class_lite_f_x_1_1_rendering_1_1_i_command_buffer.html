virtual void LiteFX::Rendering::ICommandBuffer::begin ( ) const
pure virtual

Sets the command buffer into recording state, so that it can receive command that should be submitted to the parent ICommandQueue.

Note that, if a command buffer has been submitted before, this method waits for the earlier commands to be executed by calling wait.

Implemented in LiteFX::Rendering::Backends::VulkanCommandBuffer, and LiteFX::Rendering::Backends::DirectX12CommandBuffer.

virtual void LiteFX::Rendering::ICommandBuffer::wait ( ) const
pure virtual

Waits for the command buffer to be executed.

If the command buffer gets submitted, it does not necessarily get executed straight away. If you depend on a command buffer to be finished, you can call this method.
